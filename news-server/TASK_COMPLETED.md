## 🚀 立即部署到公网

### 方案 A：Ngrok（最简单，推荐）⭐

**步骤**：

1. **下载 Ngrok**
   ```
   https://ngrok.com/download
   ```

2. **解压到任意目录**
   ```
   例如：C:\ngrok\
   ```

3. **运行部署脚本**
   ```bash
   python C:\Users\asus\.qclaw\workspace\news-server\deploy_public.py
   ```

4. **复制公网 URL**
   ```
   Ngrok 会显示：
   https://abc123.ngrok.io
   
   分享这个 URL 给其他人
   ```
