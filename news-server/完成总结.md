## 💡 工作原理

### 静态网站 + 历史数据

```
news_history.json（历史数据）
    ↓
generate_static_with_history.py（生成网站）
    ↓
index.html + news-data.js（静态文件）
    ↓
GitHub Pages（托管）
    ↓
https://xttey001.github.io/news-website/（访问）
```

### 前端交互

```javascript
// 所有交互都在浏览器中完成
// 无需后端服务器
// 完全静态，加载速度快

点击日期 → loadNews(date) → 重新渲染页面
```
