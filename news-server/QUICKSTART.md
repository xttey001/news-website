## 📖 功能说明

### 1️⃣ 新闻时间线（新功能！）

页面顶部有一个**水平时间线**，显示最近 30 天的新闻：

- **圆形按钮** = 每一天
- **月/日标签** = 日期
- **红色数字** = 该天新闻数量
- **金色高亮** = 当前选中的日期
- **点击按钮** = 查看该天的新闻

### 2️⃣ 日期选择

多种方式选择日期：

- **日期输入框** - 直接输入日期
- **快捷按钮** - 今天/昨天/上周
- **时间线按钮** - 点击任何日期

### 3️⃣ 新闻内容

每条新闻包含：

```
📰 标题（带 emoji）
├─ 📌 要点 - 一句话核心内容
├─ ⏱️ 影响时长 - 预期影响周期
├─ 📈 映射 ETF - 相关 ETF（标注利好/利空）
├─ ✨ 映射个股 - 相关个股
└─ 💡 资金信号 - 市场情绪和资金流向
```

### 4️⃣ 新闻分类

- **🔴 S 级** - 主线级新闻（结构性叙事）
- **🟡 A 级** - 轮动级新闻（阶段性叙事）
- **📱 抖音相关** - 平台相关新闻

## 🔧 与 Cron Job 集成

在你的 cron job 中添加以下代码，自动保存新闻到网站：

```python
import requests
import json

# 生成新闻数据
news_data = {
    "date": "2026-03-29",
    "market_tone": "市场基调...",
    "s_level": [...],
    "a_level": [...],
    "douyin": [...]
}

# 保存到网站
response = requests.post(
    "http://localhost:5000/api/news",
    json=news_data
)

if response.status_code == 200:
    print("✅ 新闻已保存到网站")
else:
    print("❌ 保存失败")
```

## 🐛 故障排查

### 无法访问 localhost:5000

**解决方案**：
1. 检查服务器是否正在运行
2. 查看命令行窗口是否显示 "Running on http://127.0.0.1:5000"
3. 尝试 http://127.0.0.1:5000

### 无法看到新闻

**解决方案**：
1. 检查 `news_data/` 目录是否存在
2. 运行 `python migrate.py` 恢复初始数据
3. 刷新浏览器页面

### 时间线不显示

**解决方案**：
1. 清除浏览器缓存
2. 按 F12 打开开发者工具，查看控制台错误
3. 刷新页面
