## 🔗 API 接口

### 获取新闻

```bash
GET /api/news/<date>

# 例：
curl http://localhost:5000/api/news/2026-03-29

# 返回：
{
  "date": "2026-03-29",
  "market_tone": "...",
  "s_level": [...],
  "a_level": [...],
  "douyin": [...]
}
```

### 保存新闻

```bash
POST /api/news
Content-Type: application/json

{
  "date": "2026-03-29",
  "market_tone": "市场基调",
  "s_level": [...],
  "a_level": [...],
  "douyin": [...]
}
```

### 获取所有日期

```bash
GET /api/dates

# 返回：
{
  "dates": ["2026-03-29", "2026-03-28", ...]
}
```

## 🔧 与 Cron Job 集成

在你的 cron job 中添加以下代码：

```python
import requests

# 生成新闻数据
news_data = {
    "date": "2026-03-29",
    "market_tone": "市场基调...",
    "s_level": [...],
    "a_level": [...],
    "douyin": [...]
}

# 保存到网站
response = requests.post(
    "http://localhost:5000/api/news",
    json=news_data
)

if response.status_code == 200:
    print("✅ 新闻已保存到网站")
```
