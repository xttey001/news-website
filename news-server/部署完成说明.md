## 🚀 公网访问配置（Ngrok）

### 方法 1：双击批处理文件（推荐）

1. **双击运行**：`配置Ngrok.bat`
2. 按提示注册 Ngrok 账号（免费）
3. 获取 authtoken 并粘贴
4. **双击运行**：`启动公网访问.bat`
5. 在 Ngrok 窗口中复制公网 URL

### 方法 2：手动操作

**第 1 步**：注册 Ngrok 账号
```
https://dashboard.ngrok.com/signup
```

**第 2 步**：获取 authtoken
```
https://dashboard.ngrok.com/get-started/your-authtoken
```

**第 3 步**：配置 authtoken
```powershell
C:\Users\asus\Downloads\ngrok-v3-stable-windows-amd64\ngrok.exe config add-authtoken YOUR_TOKEN
```

**第 4 步**：启动公网隧道
```powershell
C:\Users\asus\Downloads\ngrok-v3-stable-windows-amd64\ngrok.exe http 5000
```

**第 5 步**：复制公网 URL
```
在 Ngrok 窗口中找到 "Forwarding" 行
复制 https://xxxx.ngrok-free.app 格式的 URL
分享给其他人
```

## 💡 提示

### 局域网访问（无需配置）

如果你和访问者在**同一个 WiFi** 下，可以直接使用：
```
http://192.168.1.12:5000
```
无需任何配置，立即可用！

### 公网访问（需要 Ngrok）

如果访问者在**不同网络**（如不同办公室、家里），需要使用 Ngrok。

## 🔧 常见问题

### Q: Ngrok 报错 "authentication failed"？

**A**: 你需要先注册 Ngrok 账号并配置 authtoken：
1. 运行 `配置Ngrok.bat`
2. 按提示操作

### Q: 局域网访问不了？

**A**: 检查防火墙设置：
1. Windows 防火墙可能阻止访问
2. 确保服务器正在运行
3. 确认访问者和你在同一个 WiFi
