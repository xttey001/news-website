## 💡 工作原理

### 数据结构

**news_history.json**：
```json
{
  "2026-03-29": { "date": "2026-03-29", "market_tone": "...", "s_level": [...], ... },
  "2026-03-28": { "date": "2026-03-28", "market_tone": "...", "s_level": [...], ... },
  ...
}
```

**news-data.js**：
```javascript
const newsData = { ... };  // 所有新闻数据
const availableDates = ["2026-03-29", "2026-03-28", ...];  // 可用日期列表

function getNews(date) { ... }  // 获取指定日期新闻
function getPreviousDate(date) { ... }  // 获取前一天
function getNextDate(date) { ... }  // 获取后一天
```

### 前端交互

- 点击时间线上的日期 → 调用 `loadNews(date)` → 重新渲染页面
- 点击前一天/后一天 → 调用 `previousDay()` / `nextDay()` → 更新当前日期
- 输入日期框 → 调用 `loadNewsFromInput()` → 加载指定日期
