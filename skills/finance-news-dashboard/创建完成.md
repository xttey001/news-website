## 🔧 包含的脚本

### generate_static_with_history.py
**功能**：生成支持历史查看的静态网站

**特点**：
- 自动扫描 `news_data/` 目录中的所有 JSON 文件
- 每个日期都加载对应的真实新闻数据
- 生成 `index.html`（前端页面）
- 生成 `news-data.js`（新闻数据）

### daily_update.py
**功能**：日常更新流程

**特点**：
- 调用 `generate_static_with_history.py`
- 复制文件到 `github-pages-deploy/`
- 准备推送到 GitHub

### push_to_github.py
**功能**：推送到 GitHub Pages

**特点**：
- 执行 `git add/commit/push`
- 自动推送到 GitHub
- 网站 3 分钟内自动更新
