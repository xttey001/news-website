# 财经新闻推送格式规范

## 📋 核心需求

### 1. 分类标准（S/A/B）

**快速判断公式：**
```
1. 能不能影响ETF？→ 不能 = B级
2. 能不能持续1周以上？→ 不能 = B级  
3. 能不能改变资金方向？→ 能 = S级，否则 = A级
```

**分级定义：**
- **S级（L2）**：结构性叙事，能改变资金池流向，机构必须调仓
- **A级（L3）**：阶段性叙事，主线内部资金再分配
- **B级（L4）**：个股/情绪噪音，不改变资金结构

**操作原则：只做S+A，忽略B级**

---

### 2. 每条新闻的标注格式

```
【分级】标题
• 要点：一句话总结
• 时间：短/中/长
• 层级：L2/L3/L4
• ETF：映射的ETF（如有）
• 个股：映射的个股（如有）
• 链接：原文URL（如有）
```

**示例：**
```
🟥 S级 | 创新药板块拉升！集体盈利，行业拐点全面确立
• 要点：创新药行业盈利拐点确立，机构将加仓

• ETF：创新药ETF、生物医药ETF
• 个股：恒瑞医药、药明康德、百济神州
```

### 6. 去重逻辑

- 同一条新闻在3个时间段内只推送一次
- 跨天重复的新闻需要标记为"重复"或过滤
- 相同主题但不同角度的新闻视为不同条目
