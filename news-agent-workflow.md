# 财经新闻五层分析系统 - 工作流设计

## 架构概览

```
新闻输入
    │
    ▼
┌─────────────────────────────────────────────────────┐
│  第一层：🐵 悟空（新闻分析师）                        │
│  任务：深度分析新闻，判断情绪方向，提取关键变量         │
│  输出：新闻分级(S/A/B)、情绪判断、ETF/个股映射          │
└─────────────────────────────────────────────────────┘
    │
    ├──────────────────────┬──────────────────────┐
    ▼                      ▼                      ▼
┌───────────────┐  ┌───────────────┐  ┌───────────────┐
│ 第二层        │  │ 第三层        │  │ 第四层        │
│ 🐷 八戒       │  │ 🧅 沙僧       │  │ 🐴 白龙马      │
│ 贝叶斯概率    │  │ 散户情绪      │  │ 主力行为      │
└───────────────┘  └───────────────┘  └───────────────┘
    │                      │                      │
    └──────────────────────┴──────────────────────┘
                           │
                           ▼
              ┌─────────────────────────────────┐
              │ 第五层：🙏 唐僧（主Agent仲裁）    │
              │ 任务：矛盾仲裁 + 最终决策         │
              │ 输出：最终行动、仓位、风控        │
              └─────────────────────────────────┘
```

### 第五层：🙏 唐僧（主Agent仲裁）

**输入：** 所有四层输出

**核心任务：**
1. **矛盾检测**：检查四层判断是否一致
2. **矛盾仲裁**：根据规则解决冲突
3. **仓位计算**：应用仓位公式
4. **风控触发**：检查是否触发风控条件

**仓位公式：**
```
仓位 = round(八戒胜率基础仓位 × 唐僧系数 × 风控系数)
```

**唐僧系数规则：**
| 矛盾类型 | 系数 | 仓位调整 |
|---------|------|---------|
| 五层共振 | ×1.2 | 提升至80-100% |
| 逆向机会（悟空谨慎+白龙马吸筹）| ×1.15 | 提升至70% |
| 跨层矛盾（悟空看多+白龙马出货）| ×0.7 | 降至30% |
| 情绪过热（八戒高胜率+沙僧狂热）| ×0.5 | 降至35% |
| 三重危险 | ×0 | 清仓 |

**风控系数：**
- 节假日前3天：×0.7
- 地缘高度不确定：×0.8
- Trump关键发声期：×0.8

**输出格式：**
```json
{
  "矛盾检测结果": {
    "悟空vs白龙马": "一致|矛盾",
    "八戒vs沙僧": "一致|矛盾",
    "总体一致性": "高|中|低"
  },
  "仲裁决策": {
    "主要矛盾": "xxx",
    "仲裁理由": "xxx",
    "唐僧系数": 1.0
  },
  "仓位计算": {
    "八戒胜率": 72,
    "基础仓位": 50,
    "唐僧系数": 1.0,
    "风控系数": 0.8,
    "最终仓位": 40
  },
  "风控触发": [],

    "操作": "买入|卖出|观望",
```

## 调用流程（主Agent视角）

### 步骤1：调用悟空分析新闻
```javascript
sessions_spawn({
  agentId: "agent-acd13225",
  mode: "run",
  task: `分析以下新闻，输出JSON格式：

新闻标题：xxx
新闻内容：xxx
来源：xxx

输出格式要求：
{
  "level": "S|A|B",
  "sentiment": "利多|利空|中性",
  ...
}`,
  timeoutSeconds: 60
})
```

### 步骤2：并行调用八戒和沙僧
```javascript
// 悟空返回后，并行调用八戒和沙僧
sessions_spawn({ agentId: "agent-64a61d6d", task: "贝叶斯分析..." })
sessions_spawn({ agentId: "agent-217acc17", task: "散户情绪预测..." })
```

### 步骤3：调用白龙马（需要沙僧输出）
```javascript
// 等待八戒和沙僧完成后，调用白龙马
sessions_spawn({
  agentId: "agent-f189fdb5",
  task: `基于以下信息判断主力行为：
  
悟空分析：${wukongOutput}
沙僧分析：${shasengOutput}`
})
```

### 步骤4：唐僧仲裁（主Agent自己）
```javascript
// 主Agent汇总所有输出，执行仲裁逻辑
const tangDecision = arbitrate({
  wukong: wukongOutput,
  bajie: bajieOutput,
  shaseng: shasengOutput,
  bailongma: bailongmaOutput
})
```

## 测试案例

### 输入新闻：
```
标题：Trump威胁对伊朗更强打击，黄金创新高
内容：Trump连续3天在Truth Social发帖威胁对伊朗采取更强硬措施，
      黄金突破4800元/克，市场避险情绪升温。
```

### 预期流程：
1. **悟空**：S级，利多黄金，Trump反转定律触发
2. **八戒**：做多黄金概率65%（Trump喊话3天=反转概率70%）
3. **沙僧**：追高买入概率85%，FOMO情绪高涨
4. **白龙马**：主力可能在借利好出货黄金
5. **唐僧**：矛盾！悟空看多黄金但白龙马认为主力出货 + 沙僧狂热 → 降仓至30%
